<p>
Teckensnitt är uppsättningen grafiska representationer av symboler. Hur ett teckensnitt ser ut beror på typsnittets parametrar som typfamilj, typsnitt, teckensnittsproportioner etc.
Att arbeta med typsnitt är en viktig del av gränssnittsutveckling, design eller typografi. Hur ditt gränssnitt kommer att se ut beror på typsnittet du valde, men det finns också många faktorer du måste ta för att stanna vid rätt typsnitt. Inte bara användare uppmärksammar texternas läsbarhet utan även sökmotorer tar hänsyn till dem när de bildar sökresultat
</p>
<p>
 Aspose.Font för C++ är ett flexibelt och lättanvänt bibliotek för att arbeta med olika teckensnittsformat som TTF, EOT, WOFF, CFF och Type1. Med hjälp av detta API kommer du att kunna läsa olika typsnitt från filer såväl som från strömmar, tillhandahålla information om deras datastrukturer tillsammans med valfri glyf, koda information för alla teckensnittstyper och spara teckensnitt. Slutanvändare kan få hjälp genom dess renderingsundersystem för att rendera valfri glyf eller text.
</p>

     Ladda, extrahera och spara TrueType-teckensnitt - C++
    </h3>
	<pre><code class="c hljs">
    using namespace System;
    using namespace Aspose::Font::Sources;
    using namespace Aspose::Font::Ttf;
    using namespace Aspose::Font;

    // Öppna ttf-font
    SharedPtr&lt;FontFileDefinition&gt; fontFileDefinition = MakeObject&lt;FileSystemStreamSource&gt;(u"Montserrat-Regular.ttf");
    SharedPtr&lt;FontDefinition&gt; fontDefinition = MakeObject&lt;FontDefinition&gt;(FontType::TTF, fontFileDefinition);
    SharedPtr&lt;Font&gt; font = Font::Open(fontDefinition);
    
    // Woff-utgångsinställningar
    SharedPtr<IO::FileStream> outStream = IO::File::Create(u"Montserrat-Regular-new.woff");
    
    // Konvertera ttf till woff
    font-&gt;SaveToFormat(outStream, FontSavingFormats::WOFF);
